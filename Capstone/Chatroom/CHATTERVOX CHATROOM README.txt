HOW TO USE THE CHATTERVOX CHATROOM PROGRAM

1. To start the server: In a cmd window, navigate to the x64/Release folder and type "ChatroomServer.exe 54000" (the port number) and press enter.

2. To start a client window: In a new cmd window, type "ChatroomServer.exe 127.0.0.1 54000" (server IP address & port number) and press enter.
